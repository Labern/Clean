# ✦ ENHANCE

Video → 4K. Photos → beautiful. Everything stays on your Mac.

A local web app: drop an `.mp4`/`.mov`, pick a resolution (1080p / 1440p /
**4K** / 8K) and a frame rate (source / 24 / 30 / 60 / 120, with
motion-compensated interpolation), and take away a mastered HEVC file. Or flip
to IMAGE mode and drop a photo (`.jpg` `.png` `.heic` `.webp` `.tiff`) for a
denoised, upscaled, subtly graded version — hold the preview to compare with
the original.

## Run

```sh
brew install ffmpeg   # once
node server.js        # → http://localhost:2160
```

No dependencies, no accounts, no uploads to anywhere. Port 2160, because 4K.
