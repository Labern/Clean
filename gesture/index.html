▍get the real launcher on your mac

this page is the browser demo — it can't open apps. the python launcher watches your webcam in the background and fires on raised fingers: ☝ 1 = Obsidian · ✌ 2 = Claude · 🤟 3 = Spotify · 🖖 4 = this page. remap via the GESTURES table in the script or ~/.config/gesture-launcher.json — actions can be app names, URLs, or shell commands. in Terminal:

curl -O https://labern.github.io/Clean/gesture/gesture_launcher.py
pip3 install mediapipe
python3 gesture_launcher.py

macOS will ask for camera access for Terminal on first run. add --preview to see a live tracking window, or download directly: gesture_launcher.py

everything runs 100% locally — this page never uploads video anywhere (MediaPipe's model downloads once from Google's CDN).
