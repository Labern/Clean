CONVERSATION CALENDAR LIVE — CLAUDE

1. Open chrome://extensions
2. Turn on Developer mode (top-right)
3. Click "Load unpacked"
4. Select this folder
5. Go back to Conversation Calendar and press SYNC LIVE

The companion reads your conversation history from the Claude account
already signed into this browser (claude.ai). It requests no special
permissions and sends nothing anywhere — the data goes straight to the
Conversation Calendar tab and is stored only in your browser.
